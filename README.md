# Airline Flight Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a large Airline Flight Booking dataset to understand travel patterns, airline distribution, and pricing behavior across major cities in India. The analysis focuses on identifying trends using Data Visualization and Python-based Data Analysis techniques.

---

## Problem Statement
Airline Ticket Pricing is highly dynamic and influenced by factors such as airline, route, travel class, departure and arrival time, and booking window.

The objective of this project is to analyze flight booking data to identify key drivers of ticket pricing using Exploratory Data Analysis (EDA).

---

## Dataset
The dataset contains **300,000+ flight records** across major cities in India.

### Dataset Features
- Airline  
- Source City  
- Destination City  
- Departure Time  
- Arrival Time  
- Travel Class (Economy / Business)  
- Days Left Before Departure  
- Ticket Price

## Analysis & Visualizations
The following analyses and visualizations were performed:

- Airline frequency analysis  
- Departure time distribution  
- Arrival time distribution  
- Source city analysis  
- Destination city analysis  
- Ticket price comparison across airlines  
- Price comparison between Economy and Business class  
- Ticket price v/s departure time  
- Ticket price v/s arrival time  
- Route-based price analysis  
- Ticket price v/s days left before departure  

---

## Key Takeaways
- Airline choice significantly impacts ticket pricing.
- Business class tickets are considerably more expensive than economy.
- Ticket prices tend to increase as the departure date approaches.
- Certain routes and time slots influence pricing patterns.

---

## Conclusion
This analysis highlights important trends in Airline Ticket Pricing and travel patterns across major Indian cities. By applying Exploratory Data Analysis (EDA) and Data Visualization techniques, the project demonstrates how real-world datasets can be explored to understand pricing behavior and support data-driven insights in the aviation and travel domain.
